# 2026-03-30 自动化开发总结：长按配置 / 画布右键 / 右侧面板滚动

## 本轮完成

- 修复节点长按实际难以触发的问题：
  - 长按配置拆分为左键 / 中键 / 右键三项
  - 旧 `longPressAction` 配置自动迁移到左键长按
  - 左键长按不再与立即拖拽直接冲突，移动超过阈值时才转入拖拽
  - 右键长按触发动作后会抑制紧随其后的系统右键菜单
- 修复画布空白区域右键菜单：
  - 允许直接新建自由节点
  - 空白区域下无效的节点功能改为禁用状态
  - 优先级、删除、重命名、折叠、关系建立等不再错误可点
  - 菜单会在渲染后按实际尺寸重新定位，避免超出右侧或下边缘
- 修复右侧检查器面板过长问题：
  - 面板本身支持独立纵向滚动
  - 桌面布局下自动根据顶部面板和底部边距限制高度
- 修复注释输入框体验：
  - 注释框会随内容自动增长
  - 达到上限后切换为内部滚动
