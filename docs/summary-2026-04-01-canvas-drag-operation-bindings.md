# 2026-04-01 画布操作绑定改为按住拖动总结

## 完成内容

- 将画布“操作绑定”从长按触发改为按下后直接进入拖动流程，不再有长按延迟。
- 画布空白区域现在支持按绑定直接执行：
  - 拖动画布
  - 框选节点
- 右键绑定到画布操作时会抑制上下文菜单，避免和自定义画布菜单冲突。
- 画布配置字段与内部命名统一调整为 `canvas*DragAction`，不再继续使用 `LongPress` 语义。
- 保留旧版 `canvas*LongPressAction` 本地配置兼容读取，已有用户配置不会失效。
- 设置面板“操作绑定”文案改为：
  - 左键按住拖动
  - 中键按住拖动
  - 右键按住拖动
- 左键框选时补了点击抑制，避免拖出框选后又被空白画布点击清空选择。

## 验证结果

- `npm run build:web`：通过
- `go test ./...`：通过
- `npm run build:desktop`：通过
- 导出产物：`build/bin/CodeMind-1.3.5.exe`

## 版本与交付

- `wails.json` 版本号从 `1.3.4` 升级到 `1.3.5`
- 打包前检查了 `34117` 端口，占用为空，可直接打包

## 后续建议

- 若后续继续扩展画布操作，可考虑把画布绑定抽成独立枚举面板说明，进一步区分“节点手势”和“画布操作”。
