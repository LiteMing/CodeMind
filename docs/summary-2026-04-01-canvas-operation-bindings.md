# 2026-04-01 画布操作绑定与编辑延展总结

## 完成内容

- 为画布区域新增独立长按绑定：
  - 左键长按默认框选节点
  - 中键长按默认拖动画布
  - 右键长按默认无操作
- 设置面板拆分为两组：
  - `手势绑定` 仅负责节点区域动作
  - `操作绑定` 负责画布区域长按动作
- 保留旧版长按配置兼容，并将旧的 `pan-canvas` 节点长按配置迁移到对应画布按键。
- 移除画布空白区域原有的右键直接框选逻辑，改为按画布长按设置触发。
- 修正节点编辑态自适应宽度预览，编辑时保持左侧锚点不变，仅向右延展。
