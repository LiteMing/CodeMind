# 2026-03-30 自动化开发计划：节点手势 / AI 快捷交互 / 固定面板

## 目标

- 三击节点时自动触发 AI 快捷建议，默认走“建议子节点”。
- 增加 AI 快捷请求配置，允许决定是否同时请求：
  - 子节点
  - 同级节点
  - 注释
  - 连线
- 增加长按节点 AI 轮盘，支持只生成：
  - 子节点
  - 注释
  - 连线
- 补齐自由节点入口，使其与子节点 / 同级节点并列，可直接创建并自由拖拽。
- 增加操作自定义，让双击、三击、长按、空格可映射为不同动作。
- 新增“固定面板”布局；保留当前布局并命名为“浮动面板”。

## 设计

### 1. 节点手势

- 双击、三击、长按统一走“手势动作映射”。
- 默认值：
  - 双击：重命名
  - 三击：AI 快捷建议
  - 长按：AI 轮盘
  - 空格：在标题尾部编辑

### 2. AI 快捷建议

- 新增“AI 快捷请求项”开关：
  - 生成子节点
  - 生成同级节点
  - 生成注释
  - 生成连线
- 三击默认只开“子节点”，与用户当前要求保持一致。
- 后端扩展：
  - 子节点建议支持 `children / siblings` 两种模式
  - 连线建议支持 `focusNodeIds`，便于只围绕当前节点生成

### 3. 自由节点

- 新增显式 `new-floating` 命令。
- 在检查器、右键菜单、固定面板菜单里都提供入口。
- 保持自由节点不进入自动层级布局，但允许自由拖拽。

### 4. 固定面板布局

- Appearance 新增 `chromeLayout`：
  - `floating`
  - `fixed`
- `fixed` 布局在顶部显示狭长菜单栏。
- 点击 `文件 / 节点 / AI / 视图` 显示类似系统右键菜单的二级弹出菜单。

## 验证

- `go test ./...`
- `npm run build:web`
- `npm run build:desktop`
- 手工验证：
  - 三击节点是否默认触发 AI 子节点建议
  - 长按是否出现 AI 轮盘
  - 自由节点是否可直接创建并拖拽
  - 固定面板布局是否能正常切换并弹出菜单
