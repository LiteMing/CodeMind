# 2026-04-01 父子水平间距配置总结

## 完成内容

- 修复自动创建子节点和自动整理布局后，父子连线水平距离偏长的问题。
- 将父子节点的水平间距从固定常量改为可配置项 `appearance.childGapX`。
- 默认父子水平间距从原来的 `280` 调整为 `220`，使自动生成节点更靠近父节点。
- 自动创建子节点、自动创建同级节点、AI 批量生成子节点/同级节点、自动整理布局、折叠后自动整理，现已统一使用同一套间距配置。
- 在设置面板的外观区域新增“父子水平间距”数值项，修改后会立即重新整理层级布局。
- 保留旧配置兼容，不影响现有用户偏好读取。
