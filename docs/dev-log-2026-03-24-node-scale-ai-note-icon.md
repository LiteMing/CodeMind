## 图标验证结论

- `wails build -nopackage` 产出的 `build/bin/CodeMind.exe` 不会包含 Windows 资源，提取结果是系统通用图标。
- 原因已定位到本机 Wails 2.11.0 构建逻辑：只有 `Package=true` 时才会先生成 Windows `.syso` 资源。
- 完整执行 `wails build` 后，`CodeMind.exe` 已包含资源目录，提取图标与 `build/appicon.png` / `build/windows/icon.ico` 对应的自定义图标一致。
