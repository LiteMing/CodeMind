# 2026-03-30 自动化开发计划：长按配置 / 画布右键 / 右侧面板滚动

## 目标

- 修复长按未真正可用的问题，并将长按动作拆分为：
  - 左键长按
  - 中键长按
  - 右键长按
- 修复画布空白区域右键菜单：
  - 支持直接新建自由节点
  - 无效功能应禁用，不能保持可点状态
  - 菜单应根据下边缘位置回退，避免超出窗口
- 修复右侧检查器面板：
  - 面板整体支持独立纵向滚动
  - 注释输入框可随内容增长自动拉高
  - 超过上限后继续保留内部滚动

## 设计

### 1. 长按配置拆分

- `InteractionSettings` 增加：
  - `leftLongPressAction`
  - `middleLongPressAction`
  - `rightLongPressAction`
- 保留旧 `longPressAction` 作为兼容迁移来源：
  - 若旧配置存在，则迁移到左键长按
  - 中键 / 右键默认 `none`
- `handlePointerDown` 与长按状态记录按钮类型，并按按钮读取对应动作

### 2. 画布右键菜单修复

- 空白区域菜单和节点菜单分开计算可用动作
- 空白区域允许：
  - `new-floating`
- 空白区域禁用：
  - 子节点 / 同级 / 重命名 / 折叠 / 建立关系 / 优先级 / 删除
- 重新根据菜单实际高度和可视区域计算 `top`

### 3. 检查器滚动与注释自适应

- 让 `.inspector` 在桌面布局下具备独立 `overflow-y: auto`
- 约束最大高度，避免面板超出底部
- 为注释框加入自动高度同步：
  - 最小高度保留现状
  - 最大高度达到阈值后切换回滚动

## 验证

- `go test ./...`
- `npm run build:web`
- 手工验证：
  - 左 / 中 / 右键长按均按设置动作触发
  - 画布空白右键菜单能新建自由节点，其他无效项禁用
  - 右键菜单贴近底边时会上移，不超窗
  - 右侧面板可独立滚动
  - 注释框随内容增长，超上限后内部滚动
