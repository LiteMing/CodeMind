# 2026-04-02 编辑态左侧锚点预览修复总结

## 完成内容

- 修复自动宽度节点在编辑模式下再次出现的“左右同时延长”问题。
- 新增编辑态预览几何状态，记录当前编辑节点的：
  - 左侧锚点
  - 预览宽度
  - 预览高度
- 节点重新渲染时会优先使用编辑态预览尺寸，避免重渲染后回退到旧中心。
- 层级连线和关系连线在编辑态下也会使用预览中心点，因此节点会稳定表现为左侧固定、右侧延展。
- 保存中的编辑器草稿恢复流程也已同步保存和恢复预览几何，避免保存后编辑态闪回。
